About Our Project

AI-Powered Phishing Detection

This project focuses on building an AI-powered phishing link detection system. Our goal is to identify malicious and suspicious domains using machine learning models and provide an easy-to-use dashboard for monitoring threats.

The system analyzes URLs and classifies them as Safe, Suspicious, or Malicious. The website you are viewing serves as the frontend interface, allowing users to check links, view real-time analytics, and generate reports.
